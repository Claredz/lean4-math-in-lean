# MIL Progress

这个目录保存我按 *Mathematics in Lean* 章节整理后的个人学习成果。

## 这个目录和教材目录的区别

- `MIL_progress/`: 我自己的最终整理版本，准备公开展示
- `mathematics_in_lean/MIL/`: 官方教材内容和本地参考材料

## 放什么

- 我完成过的章节练习
- 我整理过的证明版本
- 我能自己解释清楚的 Lean 代码

## 不放什么

- 整套原样教材
- 只是临时参考、没有个人痕迹的内容
- 纯依赖缓存或自动生成文件

## 使用规则

- 章节命名尽量和官方 MIL 保持一致，方便对照。
- 优先保持小提交，而不是一次性堆很多改动。
- 如果内容更像自由练习，就放到 `MathProofs/`。
- 如果内容是文字总结，就放到 `notes/`。
